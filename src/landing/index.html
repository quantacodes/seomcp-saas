Frequently asked questions

What is MCP? ‚ñæ
MCP (Model Context Protocol) is an open standard by Anthropic that lets AI tools connect to external services. Think of it like USB-C for AI ‚Äî one protocol, any tool. Claude, Cursor, Windsurf, and OpenAI all support MCP. When you add seomcp.dev as an MCP server, your AI instantly gains access to all 35 SEO tools.
Do I need a Google account? ‚ñæ
Not for basic tools. Site crawling, schema validation, robots.txt analysis, and IndexNow all work without Google. To use Search Console and Analytics tools, you'll connect your Google account through a simple OAuth flow ‚Äî we never store your password, only a secure API token.
What AI tools does this work with? ‚ñæ
Any MCP-compatible client: Claude Desktop, Cursor, Windsurf, VS Code (with MCP extension), OpenAI API, and any custom MCP client. We use the standard Streamable HTTP transport, which is the official MCP remote protocol. If your tool supports MCP remote servers, it works with us.
How is this different from Ahrefs or Semrush? ‚ñæ
We use your actual Google data. Ahrefs and Semrush sell estimated metrics based on their crawlers. We connect to Google Search Console and Google Analytics directly ‚Äî so you get the exact same numbers you see in Google's own dashboards. Plus, we're $29/mo vs $129+/mo. We focus on what Google gives you for free; they focus on proprietary data like backlinks and keyword databases.
What counts as a "tool call"? ‚ñæ
Each invocation of an MCP tool counts as one call. For example, generate_report = 1 call. gsc_performance = 1 call. Session initialization and tool listing are free. The free tier gives you 50 calls/month ‚Äî enough for a couple of full audits.
Is my data secure? ‚ñæ
Yes. API keys are SHA-256 hashed (we can't read them). Google tokens are encrypted at rest. Each user gets an isolated binary process. All communication is over HTTPS. We never store your SEO data ‚Äî it flows through our server and directly to your AI tool. We're a proxy, not a database.
Can I cancel anytime? ‚ñæ
Of course. No contracts, no commitments. Cancel in your dashboard and you'll keep access until the end of your billing period. You can also downgrade to the free tier anytime.
